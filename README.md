## How It Works

The app uses Android's Canvas API to provide a smooth drawing experience. Touch events are tracked and converted into paths that are drawn on the canvas with the selected color and brush size.

## Design Philosophy

Keep it simple! The app focuses on providing the essential drawing tools without overwhelming the user with features. Perfect for quick sketches and casual doodling.
